Essential Security Practices

  • Create strong passwords(12+ characters with mix of letters, numbers, symbols)
  • Enable two-factor authentication for all accounts
  • Never use public computers for banking
  • Verify website URL before logging in (look for padlock icon)
  • Log out after every session

Fraud Prevention

  • Never share OTPs, passwords or PINs via call/SMS/email
  • Beware of phishing emails with urgent requests
  • Don't click suspiciouslinks - type bank URL directly
  • Monitor account regularly for unauthorized transactions
  • Report lost/stolen cards immediately

Digital Safety

  • Keep devices updated with latest security patches
  • Install reputable antivirus software
  • Avoid public Wi-fi for financial transactions
  • Use official bank apps from trusted stores
  • Enable remote wipe feature for mobile banking
